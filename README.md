# Amazon Lex Loan Application Chatbot

## Project Overview

This project demonstrates the development of a **Loan Application Chatbot** using **Amazon Lex**, an AI-powered service from AWS for building conversational interfaces.

The chatbot interacts with users to collect loan details, validate inputs, check eligibility, and provide loan application results.

---

## Technologies Used

* Amazon Lex (Conversational AI)
* AWS Cloud
* Natural Language Processing (NLP)
* Machine Learning (AWS managed)
* AWS Lambda (for backend logic)

---

## Key Features

* Human-like conversation interface
* Supports text and voice interaction
* Collects user information (name, age, salary, loan amount)
* Validates user inputs
* Checks loan eligibility
* Provides loan decision
* Simulates application submission

---

## How the Chatbot Works

1. User initiates conversation
2. Bot identifies intent from user input
3. Required details are collected via slots
4. Backend logic validates information
5. Eligibility is checked
6. Loan result is provided
